
# PantryOps

PantryOps is a full-stack soup kitchen management system designed to model inventory control, meal service workflows, and donation tracking using Flask, PostgreSQL, and React.

## Overview

PantryOps models the day-to-day operations of a local soup kitchen in my hometown. The system supports inventory management, meal service logging, donation intake, and role-based access control.

This project serves as a practical exploration of:

- RESTful backend architecture (Flask)
- Relational database modeling (PostgreSQL)
- Transactional workflows
- Authentication and authorization
- Continuous Integration (CI)
- Full-stack React integration
